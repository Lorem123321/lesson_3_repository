# Кое-что о `rebase`

Как быть в случае, когда точно не известно какие коммиты нужны? К счастью, Git позаботился о таких ситуациях! Можно использовать интерактивный rebase для этого - лучший способ отобрать набор коммитов для rebase.
Всё, что нужно для интерактивного rebase - это опция `-i`  
Если добавить эту опцию, Git откроет интерфейс просмотра того, какие коммиты готовы к копированию на цель rebase (target). Также показываются хеши коммитов и комментарии к ним, так что можно легко понять что к чему:

`git rebase -I HEAD~2`

перейти на 2 коммита назад от текущего положения HEAD и выбирать нужные коммиты
